gritulator.model.dc_bus._dc_bus#

DC-bus dynamic model.

The dc bus is defined as a current source with a DC capacitance model, which despicted the DC-bus voltage dynamics.

DC bus model
class gritulator.model.dc_bus._dc_bus.DCBus(C_dc=0.001, G_dc=0, i_ext=lambda t: ..., u_dc0=650)[source]#

DC bus model

This model is used to compute the DC bus dynamics, represented by a first- order system with the DC-bus capacitance dynamics.

Parameters:
C_dcfloat

DC bus capacitance (in Farad)

G_dcfloat

DC bus conductance (in Siemens)

i_extfunction

External dc current, seen as disturbance, i_ext(t).
