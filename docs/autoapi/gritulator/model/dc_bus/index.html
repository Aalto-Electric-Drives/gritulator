class gritulator.model.dc_bus.DCBus(C_dc=0.001, G_dc=0, i_ext=lambda t: ..., u_dc0=650)[source]#

DC bus model

This model is used to compute the DC bus dynamics, represented by a first- order system with the DC-bus capacitance dynamics.

Parameters:

  • C_dc (float) – DC bus capacitance (in Farad)

  • G_dc (float) – DC bus conductance (in Siemens)

  • i_ext (function) – External dc current, seen as disturbance, i_ext(t).

static dc_current(i_c_abc, q)[source]#

Compute the converter DC current, used to model the DC-bus voltage dynamics.

Parameters:

  • i_c_abc (ndarray, shape (3,)) – Phase currents.

  • q (complex ndarray, shape (3,)) – Switching state vectors corresponding to the switching instants. For example, the switching state q[1] is applied at the interval t_n_sw[1].

Returns:

i_dc – dc current (A)

float
