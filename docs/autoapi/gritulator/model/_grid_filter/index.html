gritulator.model._grid_filter#

Grid and converter filter impedance models.

This module contains continuous-time models for the AC impedance between the converter output and the AC grid. In this module, all space vectors are in stationary coordinates.

class gritulator.model._grid_filter.LFilter(U_gN=400 * np.sqrt(2 / 3), L_f=0.006, R_f=0, L_g=0, R_g=0)[source]#

Inductive-resistive filter dynamic model.

An inductive filter model is built using a simple inductance model where the two output voltages are imposed and the current can be calculated using dynamic equations. This model includes a model for an inductive-resistive impedance of the grid combined with the L-filter model.

Parameters:

  • L_f (float) – Filter inductance (H)

  • R_f (float) – Filter resistance (Ω)

  • L_g (float) – Grid inductance (H)

  • R_g (float) – Grid resistance (Ω)

class gritulator.model._grid_filter.LCLFilter(U_gN=400 * np.sqrt(2 / 3), L_fc=0.006, R_fc=0, L_fg=0.003, R_fg=0, C_f=1e-05, G_f=0, L_g=0, R_g=0)[source]#

Inductive-capacitive-inductive (LCL) filter dynamic model.

An LCL-type grid model is built using inductive and capacitive dynamic models. The two output voltages are imposed and the grid-side current, the converter-side current and the capacitance voltage can be calculated using dynamic equations. This model includes a model for an inductive-resistive impedance of the grid combined with the LCL-filter model.

Parameters:

  • L_fc (float) – Converter-side filter inductance (H)

  • R_fc (float) – Converter-side filter resistance (Ω)

  • L_fg (float) – Grid-side filter inductance (H)

  • R_fg (float) – Grid-side filter resistance (Ω)

  • C_f (float) – Filter capacitance (F)

  • G_f (float) – LCL filter conductance (S)

  • L_g (float) – Grid inductance (H)

  • R_g (float) – Grid resistance (Ω)
