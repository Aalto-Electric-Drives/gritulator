gritulator.model.ac_grid._ac_dyn_model#

Electromechanical dynamic AC grid and converter model interconnectors.

This interconnects the subsystems of a converter with a grid and provides an interface to the solver. More complicated systems could be modeled using a similar template. Peak-valued complex space vectors are used. The space vector models are implemented in stationary coordinates.
